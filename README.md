เว็บแอปพลิเคชันสำหรับผู้ที่ต้องการจัดการรถหลายคันในที่เดียว ช่วยเก็บบันทึกประวัติการซ่อมบำรุง สรุปค่าใช้จ่ายรายเดือน และมีระบบแจ้งเตือนรอบการซ่อมถัดไปอัตโนมัติผ่าน LINE เพื่อให้รถของคุณอยู่ในสภาพพร้อมใช้งานเสมอ

## ✨ Key Features
* **Fleet Management:** จัดการข้อมูลรถและอัปโหลดรูปภาพรถแต่ละคัน
* **Service Logs:** บันทึกประวัติซ่อมบำรุง เก็บข้อมูลอู่ ค่าใช้จ่าย และอัปโหลดสลิปใบเสร็จ
* **Dashboard:** สรุปค่าใช้จ่ายรวมพร้อมกราฟแสดงผลรายเดือน
* **Smart Notifications:** แจ้งเตือนรอบซ่อมล่วงหน้าผ่าน LINE Flex Message

## 🛠 Tech Stack
* **Frontend:** Next.js 14, Tailwind CSS
* **Backend:** .NET 8 Web API
* **Database & Storage:** Supabase (PostgreSQL)
* **Integrations:** LINE Messaging API, AWS Lambda (Background Jobs)
